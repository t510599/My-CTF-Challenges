Personal Web Space

Your dream space to create your own website.

Welcome to your personal web space! This is where you can upload your own web files to create your own website.

Here are instructions for uploading web files:

  1. Preparing Your Files

    • Create the web files you want to upload (HTML, CSS, JavaScript, images, etc.) on your local computer.

  2. Connecting to Login Node

    • Use FTP to connect to the login node. We have deprecated the insecure plaintext FTP.
    • Use SFTP (SSH File Transfer Protocol) to connect to login node.
      • Hostname: file.pws.chal.hitconctf.com
      • Port: 10022

  3. Creating to Your Public HTML Directory

    • Once connected, create the public_html directory within your home directory. This is where you'll upload your web files.

  4. Uploading Your Files

    • Select the files you want to upload from your local computer and upload them into the public_html directory on the server.

  5. Setting File Permissions

    • Ensure that the permissions for your uploaded files are set correctly, so our web server can fetch your files.
    • Generally, files should have permissions of 604 (-rw----r--) and directories should have permissions of 705 (drwx---r-x).
    • You can use the chmod command to adjust permissions if needed.

  6. Accessing Your Website

    • Once your files are uploaded, you can access your website using a URL constructed with service node's address and your username.
    • The URL follows this format: http://www.pws.chal.hitconctf.com/~$username/.
